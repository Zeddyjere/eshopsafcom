ESHOP WEBAPP

This project consists of a backend API and a frontend React app integrated with mpesa. 

Backend

"	Install mongoDB 
"	Open new terminal tab and enter command mongod

"	Git clone the repo 
"	cd eshop-server
"	npm install dependencies
"	nodemon / node index.js

Frontend

"	cd eshop-client
"	npm install dependencies
"	npm start
